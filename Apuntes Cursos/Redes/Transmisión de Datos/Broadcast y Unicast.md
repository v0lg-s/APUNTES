Existen 3 maneras de enviar datos desde un dispositivo a otros. Estas maneras varían entre si la información se entrega a un solo dispositivo, a algunos o a todos los dispositivos disponibles en la red.
# Unicast
La dirección MAC o IP especifica un solo dispositivo en la red.
# Multicast
La dirección MAC o IP especifica en la trama de datos un grupo de dispositivos "suscritos" para recibir la información.
# Broadcast
La dirección MAC o IP especifica que la trama de datos está dirigida a todos los dispositivos en la red.

	FF.FF.FF.FF.FF.FF
De este manera se especifica en la dirección MAC de destino que todos los hosts deben recibir la trama.
## Dominio de Broadcast
Cuando se tenga un grupo de PC's que se pueden escuchar entre sí, y sus broadcasts se dice que están dentro del dominio de broadcast.
