# Ver el uso del disco

## Comando df
Este comando nos permite ver la cantidad de espacio usado en cada dispositivo de almacenamiento.

```bash
df
```

## Comando du 
Nos permite ver un estimado del uso de espacio por ficheros.
- Tiene varias opciones, entre ellas: -h nos muestra el tama√±o en un formato legible para humanos, o -s que nos muestra solo el total de uso para el directorio que hemos especificado.

```bash
du -hs /directorio/
```
