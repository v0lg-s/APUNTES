Algunos sistemas permiten palabras clave para intervalos predefinidos:

|  **Palabra clave**  | **Equivalente en cron** |        **Descripción**        |
| :-----------------: | :---------------------: | :---------------------------: |
|       @reboot       |            -            |    Al iniciar el sistema.     |
| @yearly / @annually |       `0 0 1 1 *`       |   Una vez al año. (Enero 1)   |
|      @monthly       |       `0 0 1 * *`       |        Una vez al mes.        |
|       @weekly       |       `0 0 * * 0`       | Una vez a la semana (Domingo) |
| @daily / @midnight  |       `0 0 * * *`       |  Una vez al día (Medianoche)  |
|       @hourly       |       `0 * * * *`       |          Cada hora.           |
**Ejemplo**
```plaintext
@daily /ruta/script.sh
```
