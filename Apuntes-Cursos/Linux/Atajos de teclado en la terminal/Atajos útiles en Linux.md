# Atajos para cortar y pegar 
- **`CTRL+U`**: Corta todo lo que esté atrás del cursor.
- **`CTRL+K`**: Corta todo lo que esté frente al cursor.
- **`CTRL+Y`**: Copia lo que acabamos de cortar/eliminar.

# Atajos para moverse entre los comandos
- **`CTRL+A`**: Mueve el cursor al inicio de la línea.
- **`CTRL+E`**: Mueve el cursor al final de la línea.

# Atajo para editar algo en un archivo de texto
- **`CTRL+X+E`**: Si tenemos un comando escrito en la terminal, este atajo abrirá el editor de texto predeterminado y copiará la línea que tenemos en la terminal.
