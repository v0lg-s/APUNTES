# Ordenar lineas de un archivo de texto

## Comando sort
Este comando nos permite ordenar con diferentes opciones las lineas de un archivo de texto. Puede ser alfabéticamente de A-Z o de Z-A, dependiendo de que argumentos usemos.

Supongamos animales.txt contiene lineas así:
- Perro
- Zorro
- Ave
- Gato

```bash
sort animales.txt
```

Obtendremos un animales.txt ordenado alfabéticamente.
- Ave
- Gato
- Perro
- Zorro

Podemos consultar las opciones con "man sort".
