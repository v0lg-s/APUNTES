Linux es un sistema operativo de código abierto y gratuito que se basa en el kernel Linux. 

Sin embargo, Linux en sí mismo no es un sistema operativo completo, sino solo una parte fundamental de él.

- Un sistema operativo se compone de tres cosas:

![[Linux-kernel.png]]

- *Kernel:* Como se ve en el gráfico anterior a lo que denominamos *"Linux"* es realmente el núcleo o kernel, este kernel es el corazón del sistema operativo y se encarga de gestionar los recursos de hardware, como la CPU, la memoria, los dispositivos de entrada/salida y más. 
- *Aplicaciones:* Las aplicaciones como el explorador de archivos, reproductores, blocs de notas, etc. Pueden ser creadas por cualquier persona.
- *Sistema Operativo:* Un sistema operativo necesita tanto de aplicaciones como hardware pero necesita un medio para que estos se comuniquen, es por esto que al integrar un conjunto de aplicaciones con un Kernel se crea un sistema operativo.

Cuando se usa un conjunto de aplicaciones junto a un Kernel Linux decimos que se trata de una *Distribución* de Linux.
