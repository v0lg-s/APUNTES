Los recursos y servicios web se identifican mediante un Identificador Uniforme de Recursos (URI). Un URI es una cadena de caracteres que identifica un recurso especifico. Está compuesto por URL y URN:

- **URN (Nombre Uniforme de Recursos):** No especifica el protocolo usado por el recurso, solo identifica el espacio de nombres del recurso. (página web, documento, imagen, etc.).
- **URL (Localizador Uniforme de Recursos):** Define la ubicación de un recurso en la red, y además incluye el protocolo que se está usando para la comunicación.

# Partes de un URI
1. *Protocolo:* HTTPS u otros como FTP, SMTP, MAILTO y NNTP.
2. *Nombre de Host:* www.example.com
3. *Ruta y nombre de archivo:* /author/book.html
4. *Fragmento:*  # pagina155
