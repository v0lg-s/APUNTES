**Capa de aplicación**
SSH es la abreviatura de *"Secure Shell"*. 

- Es un protocolo que permite a una computadora conectarse de manera remota a un servidor y compartir datos de manera segura.

- SSH utiliza un canal de comunicación cifrado, por lo tanto, nadie dentro de la red podrá ver lo que se está enviando en la comunicación entre dos maquinas.

- Usa la arquitectura de cliente/host o cliente/servidor, esto permite a los usuarios conectarse a un host remotamente, es seguro porque usa un canal cifrado.
- Funciona en el puerto 22, y sustituye a Telnet que es menos seguro.

# Autenticación de usuarios

El protocolo SSH tiene un mecanismo de autenticación de usuarios que permite que dicho usuario tenga autorización para comunicarse con el servidor.
