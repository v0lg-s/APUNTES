# Web superficial
Capa a la que la mayoría accedemos.
- Contiene contenidos a los que se puede acceder utilizando un navegador web
# Web Profunda o Deep Web
Generalmente requiere autorización para ingresar a ella.
- La intranet de una organización es un ejemplo de Deep web.
# Web Oscura o Dark Web
Solo se puede acceder a esta mediante el uso de software especializado.
- Es la capa de la web preferida por los delincuentes.
- Proporciona un alto nivel de secretismo.
