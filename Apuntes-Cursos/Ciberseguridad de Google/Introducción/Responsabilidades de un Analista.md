Hay tres responsabilidades principales de un analista de seguridad de nivel de entrada:

1. *Protege computadores y sistemas de red:* El analista se encarga de monitorear la red de la organización. Generalmente, en caso de una eventualidad es el primero en responder.
	1. *Hace parte de ejercicios para probar la seguridad de la red y encontrar vulnerabilidades:* Por ejemplo pueden contribuir en pruebas de penetración.
2. *Trabajan de manera proactiva para prevenir amenazas:* Trabajan de manera conjunta con equipos de IT instalando software de prevención para identificar riesgos y vulnerabilidades.
3. *Realizan auditorias de seguridad periódicas:* Una auditoría es la evaluación y revisión de las actividades, registros y documentos de seguridad de una empresa. Cosas como que la contraseña de un equipo no esté disponible para todos los empleados, son el tipo de cosas que un analista revisa.
