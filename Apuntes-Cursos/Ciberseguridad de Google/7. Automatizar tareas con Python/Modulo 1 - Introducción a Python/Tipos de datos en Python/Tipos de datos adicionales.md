- **Tupla:** Consiste en una colección de datos como una lista, la diferencia es que esta no se puede modificar. Puede contener datos de distintos tipos.
	- **Por ejemplo,** si los identificadores de software se almacenan en una tupla para garantizar que no se alterarán, esto puede proporcionar la seguridad de que una lista de control de acceso sólo bloqueará el software previsto.
	- Se encierran entre paréntesis y se separan por comas.
	
```python
("wjaffrey", "arutley", "dkot")
(True, False, True, True)
```

- **Diccionario:** Datos que constan de un par clave-valor.
	- Los diccionarios son útiles cuando se desea almacenar y recuperar datos de forma predecible. Por ejemplo, el siguiente diccionario asigna el nombre de un edificio a un número. El nombre del edificio es el valor y el número es la clave. Después de la clave se colocan dos puntos.

```python
{   1: "East",
	2: "West",
	3: "North",
	4: "South" }
```

- **Conjunto:** Datos que consisten en una colección desordenada de valores únicos. No puede haber dos valores iguales en un conjunto. Los elementos se colocan entre llaves separados por coma.
