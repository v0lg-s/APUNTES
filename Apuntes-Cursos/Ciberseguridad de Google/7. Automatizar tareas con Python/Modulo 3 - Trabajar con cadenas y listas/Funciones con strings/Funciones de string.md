# función str()
Esta función convierte la entrada (en cualquier tipo de datos) al tipo de datos de cadena.

```python
entero = 567
cadenaEntero = str(entero) # Transforma el valor entero de 567 a una cadena "567"
```

# Función len()
Esta función retorna la cantidad de elementos de un objeto. Puede ser una lista o una cadena de caracteres.  Esto es útil a la hora de comprobar requisitos de contraseñas.

```python
largoDeCadena = len("Prueba")
print(largoDeCadena) 
# Mostrará la cantidad de letras en la palabra "Prueba". Esto es 6
```
