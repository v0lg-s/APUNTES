Para definir funciones en python se usa la palabra reservada *"def"*.
**Ejemplo:**
```Python
# Definir una función
def saludar_empleado():
	print("Bienvenido, has iniciado sesión.")
```

# Llamado de funciones
```python
# Llamar funcion
saludar_empleado()
```

**Recordar:** Una función es un trozo o sección de código que se puede reutilizar en un programa.
