# Contención
Es el acto de limitar o prevenir daño adicional causado por un incidente.
- **Ejemplo:** Desconectar de la red un equipo infectado para mitigar los daños que pueden ser causados si se propaga la infección. 
# Erradicación
Implica la completa eliminación de los elementos relacionados al incidente de todos los sistemas afectados.

Incluye acciones como:

- Realizar pruebas de vulnerabilidad
- Aplicar parches a las vulnerabilidades relacionadas a la amenaza.
# Recuperación
Es el proceso de devolver todos los sistemas afectados a sus operaciones normales.

Incluye acciones como:
- Restauración (reimaging) de los sistemas afectados.
- Reseteo de contraseñas.
- Ajustar configuración de red (reglas de firewall).
