A la hora de analizar el tráfico de una red es útil conocer los valores normales o habituales de tráfico en momentos dados. A esto se le conoce como *Línea Base*.

# Línea base
Es un punto de referencia que se utiliza para comparar. Ayudan a establecer un comportamiento esperado o normal para sistemas, dispositivos y redes. Información como:

- Direcciones IP de origen y destino
- Cantidad de datos transferidos
- Fecha y hora

Es bastante útil a la hora de crear una línea base.
