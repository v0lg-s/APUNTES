# Salting y Hash
El hash convierte la información en un código hash único que puede usarse para comprobar la integridad de los datos. Es unidireccional, es decir, el código se genera a partir de la información o datos originales pero no se puede obtener la información a partir del hash.

Salting añade caracteres aleatorios a los códigos hash haciéndolos más largos y complejos, lo que los vuelve más seguros.

# CAPTCHA y reCAPTCHA
CAPTCHA son las siglas de Completely Automated Public Turing test to tell Computers and Humans Apart (Prueba de Turing pública completamente automatizada para distinguir entre ordenadores y humanos) ayuda a evitar que programas de software intenten forzar una contraseña.

reCAPTCHA es un servicio gratuito de Google que ayuda a proteger los sitios web de bots y software malicioso.
