Un sistema operativo es la interfaz de software que media entre el hardware y las aplicaciones. Es lo que conecta las ordenes dadas por el usuario con el manejo de los recursos físicos de una computadora.

*Un Sistema Operativo vulnerable o inseguro puede comprometer la seguridad de toda una red.*

# Prácticas
## Parches de Actualizaciones
Un parche de actualización es la actualización de un software o sistema operativo que atiende vulnerabilidades de seguridad presentes en un programa o producto. 

- Cuando un fabricante publica una nueva actualización y el arreglo de vulnerabilidades específicas, indirectamente dan a los atacantes información de qué vulnerabilidad puede ser explotada en las versiones anteriores del sistema operativo.
- Es importante que las organizaciones actualicen sus sistemas operativos tan pronto como son lanzados

El sistema operativo en su versión actualizada debe ser agregado a la línea base de configuración (Baseline configuration o Baseline Image)

### Línea base de configuración
La línea base de configuración es un conjunto documentado de especificaciones dentro de un sistema que es usado como base para próximas actualizaciones, lanzamientos y construcciones. 
**Ejemplo:** Puede contener
- Las reglas de un Firewall dónde se especifican los puertos de red permitidos y no permitidos. 

## Limpieza de Hardware y Software
La correcta disposición de hardware antiguo y la eliminación de software sin uso, son buenas prácticas para endurecer la seguridad. 

- Algunos lenguajes tienen vulnerabilidades conocidas, eliminar software en desuso evita cualquier vulnerabilidad relacionada a programas que usa dicho software. 

## Implementación de políticas de contraseñas seguras
Las políticas de contraseñas fuertes requieren que las contraseñas sigan ciertas reglas, por ejemplo, un mínimo de 8 caracteres, una letra mayúscula, un número, un símbolo, etc. 
- Usualmente se define en las políticas que un usuario pierda acceso al sistema tras intentar acceder con la contraseña incorrecta cierto número seguido de veces.
- MFA (Multi-Factor Authentication)
