Practicará el uso de SQL para comunicarse con bases de datos. Aprenderá a consultar una base de datos y a filtrar los resultados. También aprenderá cómo SQL puede unir varias tablas en una consulta.

### Objetivos de aprendizaje

---

- Discuta cómo se utiliza SQL dentro de la profesión de la Seguridad.
- Describa cómo se organiza una Base de datos relacional.
- Utilice SQL para recuperar información de una base de datos.
- Aplicar filtros a las consultas SQL.
- Utilice las uniones SQL para combinar varias tablas en una consulta.
