Entre los componentes de Linux se encuentran: 
- Usuario
- Aplicaciones
- Shell
- Estándar de jerarquía del sistema de archivos/directorios 
- Kernel
- Hardware

# Usuario
Persona que interactúa con la maquina de Linux. Linux es un sistema operativo multi usuario, es decir, que más de un usuario puede usar los recursos del sistema al mismo tiempo.
# Aplicaciones
Programa que desempeña diferentes tareas.
# Shell
Es cómo nos comunicamos con el sistema.
Shell es un interprete de línea de comando. Consola.
# Estándar de jerarquía del sistema de archivos/directorios 
Es el componente del sistema operativo de Linux que organiza la información.
# Kernel
Es el componente que gestiona los procesos y la memoria. Interactúa con el hardware para ejecutar los comandos enviados por la shell. A lo que llamamos Linux es en realidad el Kernel.
- El Kernel ayuda a asegurar que los recursos sean repartidos más eficientemente y hace que el sistema funcione más rápido.
- El Kernel es el "motor" del sistema operativo Linux.
