Conocerá el sistema operativo Linux y aprenderá cómo se utiliza habitualmente en ciberseguridad. También aprenderá sobre la arquitectura de Linux y las distribuciones comunes de Linux. Además, se le presentará el shell de Linux y aprenderá cómo le permite comunicarse con el sistema operativo.

### Objetivos de aprendizaje

---

- Discuta por qué Linux se utiliza habitualmente en la profesión de la Seguridad.
- Describa la arquitectura del sistema operativo Linux.
- Identifique las características ÚNICAS de las distribuciones Linux más comunes.
- Describa cómo un shell proporciona una interfaz para que el usuario interactúe con el sistema operativo.
