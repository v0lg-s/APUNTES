Conocerá la relación entre los sistemas operativos, el hardware y el software, y se familiarizará con las funciones principales de un sistema operativo. Reconocerá los sistemas operativos comunes que se utilizan hoy en día y comprenderá cómo la interfaz gráfica de usuario (GUI) y la interfaz de línea de comandos (CLI) permiten a los usuarios interactuar con el sistema operativo.

### Objetivos de aprendizaje
---
- Describa las principales funciones de un sistema operativo.
- Reconocer los sistemas operativos más comunes en uso.
- Explique la relación entre los sistemas operativos, las aplicaciones y el hardware.
- Compare una interfaz gráfica de usuario con una interfaz de línea de comandos.
