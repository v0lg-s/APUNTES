Cuando se inicia una computadora, se activa un microchip llamado BIOS (Basic In/Out System) para computadoras más antiguas y UEFI (Unified Extensible Firmware Interface/Interfaz de Firmware Extensible Unificada) que contienen las instrucciones de carga para la computadora.

- UEFI proporciona características de seguridad mejoradas.

La última instrucción de la BIOS o el UEFI activan el cargador de arranque. Que es un programa de software que inicia el sistema operativo.

# ¿Cómo se llevan a cabo las tareas?
La realización de una tarea por parte de una computadora consta de 4 componentes:
1. **El usuario**
2. **Las aplicaciones**
3. **El sistema operativo**
4. **El hardware**

La primera parte del proceso es el usuario, este inicia el proceso teniendo algo que realizar en la computadora.

La aplicación es el programa de software con el que el usuario interactúa para completar una tarea, por ejemplo, la calculadora.

El sistema operativo recibe la solicitud hecha por el usuario a través de la aplicación. Este interpreta la solicitud y dirige su flujo. El SO envía la tarea a los componentes necesarios del hardware.

En el hardware es dónde se realiza todo el proceso para completar las tareas iniciadas por el usuario. *Ejemplo*, cuando se quiere calcular algo la CPU se encarga. Cuando se quiere guardar un archivo, el disco duro se encarga de hacerlo.

Una vez el hardware realizó la tarea, devuelve el resultado a través del sistema operativo a la aplicación para que esta pueda mostrar los resultados al usuario.
