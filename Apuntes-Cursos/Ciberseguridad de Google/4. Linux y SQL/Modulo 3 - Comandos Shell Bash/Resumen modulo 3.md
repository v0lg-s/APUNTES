Se le presentarán los comandos de Linux introducidos a través del shell Bash. Utilizará el shell Bash para navegar y gestionar el sistema de archivos y para autorizar y autenticar usuarios. También aprenderá dónde acudir en busca de ayuda cuando trabaje con nuevos comandos de Linux.

### Objetivos de aprendizaje

---

- Navegue por el sistema de archivos utilizando los comandos de Linux a través del shell Bash.
- Gestionar el sistema de archivos mediante comandos Linux a través del shell Bash.
- Describa cómo Linux gestiona los permisos de archivo.
- Utilice los comandos de Linux a través del shell Bash para autenticar y autorizar a los usuarios.
- Utilice sudo para proporcionar permisos al usuario raíz.
- Recursos de acceso que proporcionan ayuda sobre el uso de los comandos de Linux.
