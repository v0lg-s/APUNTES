Comandos ya documentados en obsidian:

- [[grep - Comando]]: Filtra de acuerdo a cadenas de caracteres en un archivo.
- [[Operador pipe]]: Un operador que envía del standard output de un comando al standard input de otro. Funciona como una tubería con dos extremos. 
- [[Find - Comando]]: busca directorios y archivos que cumplan los criterios especificados. Existe una amplia gama de criterios que pueden especificarse con find. 
	- Cuando utilice find, el primer argumento después de find indica dónde empezar a buscar. Por ejemplo, si introduce find /home/analyst/projects buscará todo a partir del directorio projects.
	- Después de este primer argumento, debe indicar sus criterios para la búsqueda. Si no incluye un criterio de búsqueda específico con su segundo argumento, es probable que su búsqueda devuelva un montón de directorios y archivos.
