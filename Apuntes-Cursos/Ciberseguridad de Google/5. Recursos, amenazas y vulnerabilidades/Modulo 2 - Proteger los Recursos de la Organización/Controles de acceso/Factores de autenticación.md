1. **Conocimiento:** Algo que el usuario sepa.
2. **Propiedad:** Algo que el usuario posee. (Puede ser su teléfono, se envía un código al dispositivo que se tiene certeza el usuario posee)
3. **Característica:** Algo que el usuario es. (Biometría)

**Autenticación Multi Factor**
Es una medida de seguridad que requiere que un usuario verifique su identidad de dos o más maneras para acceder a un sistema o red.
