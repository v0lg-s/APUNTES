Es un protocolo de autorización de estándar abierto, que comparte el acceso designado entre aplicaciones. Usa tokens para establecer que está autorizado para acceder.

Por ejemplo un servicio puede usar la cuenta de Google para crear su cuenta, el servicio usa estos tokens para saber si el usuario está autorizado. 

# API Token
Es un pequeño bloque de código encriptado que contiene información acerca del usuario.
