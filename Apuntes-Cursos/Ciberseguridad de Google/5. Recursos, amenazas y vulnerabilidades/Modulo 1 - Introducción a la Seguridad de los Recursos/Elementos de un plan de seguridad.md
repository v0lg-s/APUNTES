# Tipos de categorías de riesgos
Algunas de los tipos de categorías de riesgo son:
- Daño
- Divulgación
- Pérdida de información

# Elementos de un plan de seguridad
- *Políticas:* Conjunto de reglas que reducen el riesgo y protegen la información.
- *Estándares*: Puntos de referencia que informan cómo adoptar las políticas. 
- *Procedimientos*: Instrucciones paso a paso que desempeñan una tarea de seguridad específica.
