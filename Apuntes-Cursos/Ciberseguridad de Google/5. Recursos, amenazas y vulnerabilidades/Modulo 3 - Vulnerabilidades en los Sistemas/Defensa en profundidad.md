Es un acercamiento a la gestión de vulnerabilidades en forma de capas que reduce el riesgo. 

Esta estrategia trata de defender en diferentes etapas o capas los recursos, si un atacante sobrepasa una capa es recibido por otra capa de defensa.

Las capas de esta estrategia son:

# 1. Capa perimetral
Es principalmente una capa de autenticación de usuarios que filtra el acceso externo. Incluye tecnologías como:
- Nombres de usuario
- Contraseñas

# 2. Capa de red
Está mas alineada a la autorización. Incluye tecnologías como:
- Firewalls

# 3. Capa de endpoints o dispositivos finales
Protege servidores, laptops, pc's etc. 
Incluye tecnologías como Software antivirus

# 4. Capa de aplicación
Incluye todas las interfaces usadas para interactuar con la tecnología. Las medidas de seguridad aquí son programas como parte de la aplicación.
- Autenticación de múltiple factor.

# 5. Capa de datos o información
Información crítica que debe ser protegida, como PII o SPII.
Clasificación de recursos es una de las medidas en esta capa.
