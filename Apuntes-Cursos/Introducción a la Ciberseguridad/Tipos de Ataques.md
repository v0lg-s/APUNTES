# Denegación de Servicio(DoS)

DoS, es un tipo de ataque de red relativamente sencillo de llevar a cabo. Da como resultado una interrupción en el servicio de red a los usuarios, dispositivos o aplicaciones objetivo.

==Se hace haciendo llegar una cantidad abrumadora de tráfico a la red o enviando paquetes maliciosamente formateados.==

# DoS Distribuido (DDoS)

Es similar al DoS pero proviene de múltiples fuentes coordinadas.

*Ejemplo:* Mediante el uso de Botnets el atacante coordina a todos los sistemas para hacer peticiones a un servicio específico.

**Botnets:** Es un grupo de bots conectados mediante internet que pueden ser controlados por un individuo o grupo. 
- Una computadora bot puede ser infectada por ingresar a un sitio web malicioso o abrir un archivo infectado. 
- Se pueden usar para distribuir malware, lanzar ataques DDoS, distribuir spam o ejecutar ataques de fuerza bruta.

# Ataques en el Camino

Se conocen también como ataques de Man In the Middle(MitM) o Man in the Mobile(MitMo).

1. *MitM:* El atacante toma control de un dispositivo sin que el usuario lo sepa, puede interceptar y capturar información saliente y entrante.
2. *MitMo:* En este tipo de ataque se interceptan o modifican las comunicaciones entre dos dispositivos como un navegador y un servidor.
# Envenenamiento SEO

Se utiliza para posicionar un sitio web entre los primeros tras realizar una búsqueda en motores de búsqueda como Google.

# Ataques de Contraseña

1. *Pulverización de Contraseña:* Se intenta obtener acceso a un sistema "rociando" contraseñas en distintas cuentas.
2. *Ataques de Diccionario:* Se intentan sistemáticamente todas las palabras contenidas en un diccionario de contraseñas comunes.
3. *Ataques de Fuerza Bruta:* El atacante intenta acceder probando todas las combinaciones posibles de letras, números y símbolos hasta que accede.
4. *Ataque de Arcoíris:* Se usan tablas con contraseñas y sus hashes calculados, el atacante compara los hashes, desde una base de datos que ha comprometido previamente, y si encuentra una coincidencia en la lista ya podrá conocer la contraseña en texto claro. 

**Programas que ejecutan ataques de fuerza bruta:** OphCrack, THC Hydra, L0phtCrack, rainbowCrack, Medusa.
