
1. En primer lugar el comando *"git init"* crea un nuevo repositorio de Git. Puede utilizarse para convertir un proyecto existente y sin versión en un repositorio de Git, o para inicializar un nuevo repositorio vacío.  
	1. Este comando creará un directorio oculto *.git* dentro del directorio en el que lo ejecutamos. Este directorio contiene toda la información necesaria para el proceso de crear un repositorio.
2. El comando *"git add"* seguido de los archivos que queremos enviar al repositorio, hará que se agreguen a la fase stage.
3. El comando *"git commit -m "nombre-commit" "* hará que los archivos agregados en el paso anterior pasen a la etapa de commit.
