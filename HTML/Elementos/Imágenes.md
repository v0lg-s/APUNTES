La etiqueta para introducir imágenes en HTML es:
```html
<img>
```
Es una etiqueta de auto cerrado es decir no es necesario una etiqueta de cerrado para contener información de la imagen en cuestión. (Self Closing Tag)

# Atributos
- *src (source):* Contiene la ruta de la imagen.
- *alt (alternative):* Muestra un texto alternativo cuando la imagen no carga correctamente.

*Ejemplo:*
```html
<!DOCTYPE html>

<html>
    <head>
        <title>Imágenes</title>
    </head>

    <body>
        <img src="recursos/lobo.jpg" alt="Imagen de un lobo">
    </body>
</html>
```
