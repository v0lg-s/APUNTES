Hay muchos directorios dentro de *"/inhere"* con demasiados archivos, hay que automatizar la búsqueda.

Con el comando *find* se puede realizar una búsqueda recursiva desde el directorio especificado. Entonces el primer parámetro de nuestro comando será *"."* para indicar que el directorio de inicio será el actual.

```bash
find .
```

Sabemos que se trata de un archivo. Opción *-type* y el argumento *f* para indicar *"file"*

```bash
find . -type f
```

Comencemos con las características.

1. Es legible: Opción *-readable* para encontrar archivos con permiso de lectura para el usuario que ejecuta el comando.
2. No ejecutable: Opción *-executable* indica que es ejecutable, la negamos con *"!"*.
3. Tamaño de 1033 bytes: Opción -size para indicar tamaño añadimos una "c" al final del tamaño para indicar que se trata de bytes.

```bash
find . type -f -readable ! -executable -size 1033c
```
