La flag está en un archivo en algún lugar del servidor **(todo)** con las siguientes características:

- Su propietario es bandit7
- El grupo propietario es bandit6
- Tiene 33 bytes de tamaño

# Solución

- Para encontrar un archivo con estas características uso el comando *"find"*. 
- Dado que la búsqueda se debe realizar en todo el servidor el primer parámetro será la raíz:

```bash
find /
```

Esto iniciara la búsqueda desde el directorio raíz del servidor.

Hay opciones que nos permiten filtrar por las características antes mencionadas:

* **Opcion -user:** Le indicamos el nombre de usuario a quien pertenece el archivo.
* **Opcion -group:** Le indicamos el nombre del grupo a quien pertenece el archivo.
* **Opcion -size:** Le indicamos el tamano del archivo con la opcion "c" que indica bytes.

 ```bash
find / -user bandit7 -group bandit6 -size 33c
```

Este comando dará como salida algunos errores de permiso denegado. Podemos limpiar la salida enviado el error estándar [[STDERR - Standard Error|STDERR]] a *"/dev/null"*.

 ```bash
find / -user bandit7 -group bandit6 -size 33c 2>/dev/null
```
