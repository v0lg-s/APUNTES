La flag está en un archivo con múltiples cadenas ilegibles.

Podemos usar el comando *"strings"* para que se nos muestren las secuencias de caracteres legibles o imprimibles.

```bash
strings data.txt
```

Esto nos mostrara varias cadenas en líneas de texto.

Se menciona que la flag esta precedida de varios caracteres *'='*. 

Podemos filtrar con el comando *'grep'* de dos maneras:
1. Filtrando por la cadena *'\=\=\=\='* de *x* caracteres *'='* seguidos.

```bash
grep "===="
```

2. O con la *expresión regular* que indica que lo que buscamos esta precedido de *x* numero de iguales.

```bash
grep -E '=+' o grep -E '={2}
```

```bash
strings data.txt | grep -E '={2}'
```
