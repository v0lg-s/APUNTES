La contraseña de este nivel se consigue ejecutando un script llamado *'./suconnect'*, su funcionamiento es el siguiente:

1. Recibe como argumento un número de puerto.
2. Se conecta al puerto especificado usando TCP.
3. Lee de la conexión y si recibe la contraseña de bandit 20, devuelve la contraseña de bandit 21 en respuesta.

Debemos poner la maquina en escucha por un puerto cualquiera x, usando netcat.

```bash
nc -l 45656
```
 
 Se pone en escucha por el puerto 45656

Es necesario que desde este puerto se envíe la contraseña de *bandit 20* cuando se establezca una conexión.

Lo realizamos con el siguiente comando:

```bash
nc -l 45656 < 0qXahG8ZjOVMN9Ghs7iOWsCfZyXOUbYO
```

```bash
nc -l 45656 < /etc/bandit_pass/bandit20
``` 

 (Enviamos la cadena a la conexión como entrada)
 
Es necesario poner este proceso en segundo plano (Volverlo un demonio/daemon) para poder crear la conexión ejecutando el script *'./suconnect'*.

Usamos el símbolo *'&'* al final del comando para ejecutarlo en segundo plano. Útil revisar: [[1. Qué es el control de trabajos de Unix]]

```bash
nc -l 45656 < /etc/bandit_pass/bandit20 &
``` 

Ahora podemos ejecutar el script

```bash
./suconnect 45656
```

La conexión nos dará la contraseña en donde sea que el comando de escucha se esté ejecutando.

