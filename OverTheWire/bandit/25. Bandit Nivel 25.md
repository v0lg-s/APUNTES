Hay un demonio escuchando en el puerto *30002* y nos dará la contraseña si se le envía la contraseña de *bandit 24* junto a un pin de 4 dígitos. Debemos probar con fuerza bruta todas las posibles combinaciones.

Creé el script:

```bash
#!/bin/bash

for i in {0000..9999}; do
        echo "gb8KRRCsshuZXI0tUuR6ypOFjiZbf3G8 $i"
done | nc localhost 30002
```

Envía la clave de bandit 24 + el código de 4 dígitos. Se usa *'0000'* en el rango del bucle for, para indicar que se deben generar números de 4 dígitos.

Se envía mediante pipe la salida del bucle for al socket  *localhost:30002*.

El demonio responde con:

```plaintext
Correct!
The password of user bandit25 is iCi86ttT4KSNe1armKiwbQNmB3YJP3q4
```
