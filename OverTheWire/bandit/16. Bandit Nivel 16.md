Para encontrar la contraseña de bandit 16 debemos enviar la contraseña de *bandit 15* al *puerto 30001 en localhost*.

- `localhost:30001` **Socket**

Lo debemos hacer usando encriptación SSL/TLS, para esto usamos la herramienta *'openssl'* con el subcomando *'s_client'*.

*Openssl* es una herramienta para realizar procedimientos de criptografía.

El comando *s_client* se usa para simular un cliente SSL/TLS genérico, que funciona para establecer conexiones de prueba con un servidor usando la encriptación SSL/TLS.

```bash
openssl s_client localhost:30001
```
