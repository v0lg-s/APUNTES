La contraseña está en un archivo llamado *data.txt*, este archivo es el hexdump de un archivo que comprimieron varias veces.

Usé el comando '*xxd'* para tratar con hexdumps.

La opción *'-r'* revierte el hexdump al archivo original.

```bash
xxd -r data.txt > data
```

Usé el comando *'file'* para saber que tipo de archivo era *'data'*.

* **El resultado es:** Un archivo comprimido con *gzip*, es decir, su formato o extensión es *.gz*. Cambié el nombre incluyendo la extensión.

```bash
file data
mv data data.gz
```

* Con el nombre del archivo ya modificado, lo que hice fue leer el manual de *gzip*. Para descomprimir archivos *.gz* se usa el comando *'gzip'* con la opción *'-d'* junto al nombre del archivo.

```bash
gzip -d data.gz
```

* El resultado queda en un archivo llamado *data*. Se repite el proceso para revisar el tipo de compresión hecha.
