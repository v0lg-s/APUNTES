# Método replace()
Este método se usa para cambiar los valores de año, mes o día de un objeto de tipo `date`, acceder a estos atributos directamente, no es posible, ya que estos son atributos de solo lectura.

```python
from datetime import date

d = date(1991, 2, 5)
print(d)

d = d.replace(month=1, day=16)
print(d)
```

# Método weekday()
El método `weekday()` devuelve el día de la semana de una fecha específica, como un número entero, donde 0 es lunes y 6 es domingo.

```python
from datetime import date

d = date(2019, 11, 4)

print(d.weekday()) # fue un lunes, devuelve 0
```

El método `isoweekday()` hace lo mismo pero el retorno es 1 para lunes y 7 para domingo.
