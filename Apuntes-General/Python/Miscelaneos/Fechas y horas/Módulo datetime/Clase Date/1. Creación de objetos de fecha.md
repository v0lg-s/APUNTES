En el módulo `datetime` una de las clases proporcionadas es una clase llamada `date`.

Los objetos de esta clase representan una fecha que consta de año, mes y día.

Método `today()` devuelve un objeto de tipo `date` que representa la fecha actual.

```python
from datetime import date

today = date.today()

print("Hoy: ",today)
print("Año: ", today.year)
print("Mes: ", today.month)
print("Día: ", today.day)

```

# Crear nuestro propio objeto `date` 

Se deben pasar los parámetros al constructor.

```python
from datetime import date

my_date = date(2019, 11, 4) # año, mes, dia
```
