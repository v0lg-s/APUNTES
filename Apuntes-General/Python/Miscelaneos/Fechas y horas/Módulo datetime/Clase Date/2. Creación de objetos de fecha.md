# Creación a partir de un timestamp

Una marca de tiempo es algún dato que indica cuándo pasó algo.

Usualmente al decir timestamp se refieren a la *Unix Epoch timestamp* que son los segundos que han pasado desde el 1 de enero de 1970 00:00:00 (UTC). 

Se usa esta fecha para calcular una marca de tiempo y se puede usar para generar un objeto de fecha, de la clase `date`.

```python
import time
from datetime import date

timestamp = time.time() # devuelve la Unix Epoch timestamp

d = date.fromtimestamp(timestamp)
print("Fecha:", d)
```


# Creación usando el formato ISO

El estándar ISO 8601 define las fechas en el formato **AAAA-MM-DD**, el método de la clase `date` `fromisoformat` que toma un string con una fecha en este formato y devuelve un objeto `date`.

```python
from datetime import date

fecha = date.fromisoformat("1990-10-01")

print(fecha)
```
