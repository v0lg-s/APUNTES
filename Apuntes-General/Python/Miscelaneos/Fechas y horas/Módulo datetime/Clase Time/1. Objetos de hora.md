El módulo `datetime` tiene una clase llamada `time` que permite presentar la hora.

```python
time(hour, minute, second, microsecond, tzinfo, fold)
```

|   Parámetro   |                    Restricciones                    |
| :-----------: | :-------------------------------------------------: |
|    `hour`     |     Debe ser mayor o igual que 0 y menor que 23     |
|   `minute`    |     Debe ser mayor o igual que 0 y menor que 59     |
|   `second`    |     Debe ser mayor o igual que 0 y menor que 59     |
| `microsecond` | Debe ser mayor o igual que 0 y menor que 1.000.000  |
|   `tzinfo`    | Debe ser un objeto de la subclase `tzinfo` o `None` |
|    `fold`     |          Debe ser 1 o 0 (predeterminado 0)          |

```python
from datetime import time

t = time(14, 53, 20, 1)

print("Tiempo:", t)
print("Hora:", t.hour)
print("Minutos:", t.minute)
print("Segundos:", t.second)
print("Microsegundo:", t.microsecond)
```
