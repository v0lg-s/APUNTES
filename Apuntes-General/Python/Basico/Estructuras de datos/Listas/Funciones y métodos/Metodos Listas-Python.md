
---
## Agregar elementos

```python
lista.append(value)
```

Agrega el elemento *'value'* al final de la lista.

```python
lista.insert(location, value)
```

Agrega el elemento especificado a la lista en la ubicación "location".

---
## Contar elementos
```python
lista.count(value)
```

Cuenta el número de repeticiones del elemento *'value'*.

---
## Mostrar índice
```python
lista.index(value)
```

Retorna el índice de la primera aparición del elemento **'value'**.

---
## Quitar elementos
```python
lista.remove(value)
```

Remueve la primera aparición del elemento **'value'**.

---
## Ordenar elementos

```python
lista.sort()
```

Ordena los elementos de la lista de forma ascendente.

---
## Invertir orden

```python
lista.reverse()
```

Invierte el orden de los elementos en la lista. útil con sort para ordenar de manera descendente.
