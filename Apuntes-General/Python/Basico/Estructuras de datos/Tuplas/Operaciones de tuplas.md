
- El operador `+` puede unir tuplas.
- El operador `*` multiplica tuplas, como en las listas.
- `in` y `not in` funcionan igual que en las listas.

*Ejemplo*
```python
tupla = (1, 10, 100)

tupla_adicional = tupla + (1000, 1000)

tupla_por_3 = tupla * 3

```

`tupla_adicional:` (1, 10, 100, 1000, 10000)
`tupla_por_3:` (1, 10, 100, 1, 10, 100, 1, 10, 100)
