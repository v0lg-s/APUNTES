Genera una secuencia de números que puede ir desde 0 por defecto, u otro inicio dependiendo del primer argumento.

```python
range(inicio,final,paso)
```

**Nota:** Tener en cuenta que el número del argumento "final" no se incluye.

*Ejemplo:*
```python
range(1,9,2)
```
Generará la secuencia: (1,3,5,7)
