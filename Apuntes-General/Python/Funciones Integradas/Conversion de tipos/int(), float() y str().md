
# int
Toma un argumento e intenta convertirlo a un valor entero.

```python
anything = int(input("Ingresa un número: "))
something = anything ** 2.0
print(anything, "a la potencia de 2 es", something)
```

# float
Toma un argumento e intenta convertirlo a un valor flotante.
```python
anything = float(input("Ingresa un número: "))
something = anything ** 2.0
print(anything, "a la potencia de 2 es", something)
```
