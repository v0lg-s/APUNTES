Existen rutas absolutas y Relativas.

Cuando redirigimos a otros sitios web usamos rutas absolutas. Sin embargo, en nuestro propio sitio web o proyecto debemos usar rutas relativas para evitar que los enlaces se rompan al moverse el proyecto a otros lugares
# Rutas Absolutas
Las rutas absolutas representan la dirección absoluta de un elemento, es decir, independientemente de la ubicación del recurso con una ruta absoluta se puede llegar.

Estas rutas absolutas parten del directorio raíz hasta llegar al recurso concreto que se está buscando.

# Rutas Relativas
Estas rutas tienen en cuenta el directorio desde el que se está invocando el recurso, es decir, realizará la búsqueda desde el directorio actual.
