# Encabezados 
La etiqueta para definir un encabezado es:
```html
<body>
	<h1>Encabezado</h1>
</body>
```

El número indica la jerarquía de cada encabezado, entre más grande es el número menor jerarquía/altura/tamaño tiene el encabezado. *Por ejemplo:*

![[Encabezados.png.png]]

En los documentos HTML es buena práctica usar el encabezado h1 solamente una vez para describir en detalle la página web. El posicionamiento SEO penaliza usar múltiples veces h1.
