Son listas en las que importa el orden en que los elementos estén organizados. Por ejemplo una sección de top de películas en una página web.

La etiqueta usada para describir este tipo de listas es:
```html
<!DOCTYPE html>
<html>
	<head>
        <title>PRUEBA</title>
    </head>
	<body>
		<ol>
			<li>Elemento 1</li>
			<li>Elemento 2</li>
			<li>Elemento 3</li>
			<li>Elemento 4</li>
		</ol>
	</body>
</html>
```
Siendo 
```html
<ol> Ordered List
<li> List Item
```
