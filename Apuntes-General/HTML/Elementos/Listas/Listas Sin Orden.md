Son listas en las que NO importa el orden en que los elementos estén organizados. Por ejemplo una barra de navegación con opciones.

La etiqueta usada para describir este tipo de listas es:
```html
<!DOCTYPE html>
<html>
	<head>
        <title>PRUEBA</title>
    </head>
	<body>
		<ul>
			<li>Elemento 1</li>
			<li>Elemento 2</li>
			<li>Elemento 3</li>
			<li>Elemento 4</li>
		</ul>
	</body>
</html>
```
Siendo 
```html
<ul> Unordered List
<li> List Item
```
