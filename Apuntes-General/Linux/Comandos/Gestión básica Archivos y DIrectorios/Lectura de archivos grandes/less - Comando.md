## Comando less
El comando less muestra el contenido de un archivo página a página. Esto permite avanzar o retroceder fácilmente por el contenido.
- `space bar`: Avanza una página.
- `b`: Retrocede una página.
- `Down Arrow`: Avanzar una línea.
- `Up Arrow`: Retroceder una línea.
- `q`: Salir y volver a la ventana de terminal anterior.
