- **Comprobar o listar las tareas cron activas:**

```bash
crontab -l
```

- **Editar las tareas cron del usuario que lo ejecuta:**

```bash
crontab -e
```

- **Eliminar tareas cron:**

```bash
crontab -r
```

# Comandos del servicio/demonio cron

```bash
sudo systemctl start cron       # Iniciar el servicio
sudo systemctl stop cron        # Detener el servicio
sudo systemctl restart cron     # Reiniciar el servicio
sudo systemctl status cron      # Verificar el estado del servicio
```
