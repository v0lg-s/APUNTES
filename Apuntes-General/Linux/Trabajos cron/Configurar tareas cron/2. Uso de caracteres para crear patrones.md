# Caracteres especiales 
## Asterisco ( * )
El asterisco significa 'todos los valores posibles para este campo'

**Por ejemplo:**

```plaintext
* * * * * /ruta/script.sh
```

Esto ejecuta el script cada minuto de cada hora, todos los días, todos los meses.

## Separador coma ( , )
Permite enumerar valores específicos para un solo campo.
**Ejemplo:**
```plaintext
0,15,30,45 * * * * /ruta/script.sh
```

Se ejecuta a los minutos 0, 15, 30 y 45 de cada hora.
## Guion (-)
Permite marcar rangos en un campo.
**Ejemplo:**
```plaintext
0 9-17 * * * /ruta/script.sh
```

Se ejecuta desde las 9am hasta las 5pm cada hora.

## Barra inclinada ( / )
Permite especificar intervalos de valores.

**Ejemplo:**
```plaintext
*/5 * * * * /ruta/script.sh
```

Se ejecuta cada 5 minutos.

**Ejemplo:**
```plaintext
* 0-12/2 * * * /ruta/script.sh
```

Se ejecuta de media noche a medio día en intervalos de 2 horas.
