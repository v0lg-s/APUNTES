# Lectura de entrada del usuario
Se usa la palabra reservada *"read"* para captura la entrada del usuario.

```bash
read -p "Ingresar edad: " edad
echo "Tienes $edad años."
```

La opción -p muestra en placeholder para la entrada de datos. 

En el ejemplo anterior `read` asigna la entrada del usuario a la variable `edad`.

# Salida
El comando 'echo' muestra mensajes.
```bash
echo "Directorio personal: $HOME"
```
